# Stone Paper Scissors Game 🎮

## Project Description
This is a beginner-level mini project of the classic Stone Paper Scissors game built using HTML, CSS, and JavaScript. The project demonstrates basic web development skills such as structuring a webpage, styling it with CSS, and adding interactivity using JavaScript.

## Features
- Simple and interactive game interface
- User can choose Stone, Paper, or Scissors
- Computer generates a random choice
- Displays the winner of each round
- Score tracking for user and computer

## Technologies Used
- HTML
- CSS
- JavaScript

## How the Game Works
1. The player selects Stone, Paper, or Scissors.
2. The computer randomly selects one option.
3. The game compares both choices.
4. The result (Win, Lose, or Draw) is displayed.
5. Scores are updated accordingly.
